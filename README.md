Add meta data to `<head>` for your [gitbooks](https://www.gitbook.com/).

## Config

In your gitbook.json, add this plugin:

```javascript
{
    "plugins": [
        "meta-single"
    ]
}
```

And add/edit the config:

```javascript
{
    "plugins": [
        "meta"
    ],
    "pluginsConfig": {
        "meta-single": {
            "name": "name",
            "content": "content"
        }
    }
}
```
